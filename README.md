# 📄 txt2pdf 

**txt2pdf** — это элегантный и минималистичный веб-инструмент для мгновенной конвертации текстовых файлов в профессиональные PDF-документы. Дизайн вдохновлен эстетикой *Arena.ai*, сочетая в себе глубокие темные тона, теплую палитру и безупречную типографику.

## ✨ Особенности

*   **🎨 Эстетика Arena.ai:** Утонченный интерфейс в кофейно-антрацитовых тонах с использованием шрифтов *Playfair Display* и *Inter*.
*   **🔒 Приватность прежде всего:** Конвертация происходит полностью на стороне клиента (в браузере). Ваши данные никогда не покидают компьютер.
*   **🔡 Полная поддержка UTF-8:** Корректное отображение кириллицы и спецсимволов благодаря встроенному шрифту Roboto.
*   **⚙️ Гибкая настройка:**
    *   Регулировка размера шрифта (6–24 pt).
    *   Настройка полей страницы (5–40 мм).
    *   Опциональная нумерация строк для кода или документов.
    *   Автоматические колонтитулы с именем файла и номером страницы.
*   **🚀 Zero Setup:** Не требует установки. Один HTML-файл содержит всё необходимое.

---

## 🛠 Технологический стек

Инструмент построен на современных и надежных технологиях:
*   **jsPDF** — мощный движок для генерации PDF в браузере.
*   **Font Awesome 6** — изящные векторные иконки.
*   **Google Fonts** — премиальная типографика.
*   **CSS3 Custom Properties** — для глубокой настройки цветовой палитры.

---

## 🚀 Быстрый старт

1.  Скачайте файл `index.html`.
2.  Откройте его в любом современном браузере.
3.  Перетащите ваш `.txt`, `.py`, `.js` или другой текстовый файл в зону загрузки.
4.  Настройте внешний вид и нажмите **«Конвертировать»**.

---

## 🛡 Безопасность и защита

*   **No Server-side:** Приложение не имеет бэкенда. Это гарантирует отсутствие утечек данных.
*   **Copy Protection:** Интерфейс защищен от случайного выделения и копирования элементов управления для обеспечения чистого пользовательского опыта.
*   **Input Validation:** Встроенная система контроля вводимых значений предотвращает ошибки при генерации PDF.
